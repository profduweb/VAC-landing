# VAC — Landing page

Landing page statique de **Vulgarisateur Apple & Cie**, prête pour GitHub Pages.

## Publication

1. Envoyer tous les fichiers à la racine du dépôt GitHub.
2. Dans **Settings → Pages**, choisir **Deploy from a branch**.
3. Sélectionner la branche `main` et le dossier `/ (root)`.

Le site utilise actuellement cette URL canonique :

`https://appleetcompagnie.com/`

Si l’adresse finale est différente, remplacer cette URL dans :

- `index.html`
- `robots.txt`
- `sitemap.xml`

## Fichiers principaux

- `index.html` — contenu, SEO, Open Graph et données structurées
- `styles.css` — design responsive
- `script.js` — lecteur YouTube chargé au clic
- `social-card.jpg` — carte de partage 1200 × 630
